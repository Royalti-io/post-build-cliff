The Cliff

Every "build an app in 20 minutes" tutorial ends where this module begins. Here's what the post-build cliff actually looks like — and how to tell you're standing near the edge.

What you'll walk away with
  • Read and write a useful .gitignore
  • Tell a private repo from a public one
  • Spot which of your keys are secrets
  • Know what a staging environment is for
  • Run a recurring data export
  • Recognise the five most common "11pm deploy" failures
